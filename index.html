📊 Curso: Excel aplicado a negocios

¡Aprende a dominar Excel con casos reales y ejercicios prácticos!

🌟 ¿Por qué deberías tomar este curso?

Este curso está diseñado especialmente para quienes desean aprender Excel desde cero, sin necesidad de conocimientos previos.
A lo largo de las sesiones, avanzarás desde lo más básico hasta un nivel intermedio, combinando teoría y práctica con ejemplos aplicados al mundo de los negocios.📈
Ya sea que manejes un emprendimiento, una tienda, prestes servicios o simplemente quieras ordenar tus finanzas, aprenderás a usar Excel como una herramienta poderosa para mejorar tu gestión, tomar decisiones basadas en datos y optimizar tu tiempo.
Cada clase está pensada para ser clara, útil y 100% práctica. 🚀

📘 Curso ⏳ Duración 🗓 Fechas 📅 Días 🕐 Horario
Excel aplicado a negocios 6 sesiones 7, 9, 11, 21, 23, 25 de abril lunes, miércoles y viernes 17:30 a 19:00

📝 Contenido del Curso

📌 Sesión 1: Introducción y Fundamentos de Excel

  • ¿Qué es Excel y para qué sirve?
  • Interfaz y funciones básicas
  • Tipos de datos: texto, números y fechas
  • Edición básica: insertar, eliminar y mover celdas
  • Ejercicio práctico: Introducción de datos y primeros pasos

🧩 Sesión 2: Gestión y Organización de Datos

  • Celdas, filas y columnas: gestión y edición avanzada
  • Ordenar y filtrar datos
  • Creación de listas y validación de datos
  • Ejercicio práctico: Organización de datos y creación de una lista de inventario simple

🧮 Sesión 3: Fórmulas y Funciones Básicas

  • Introducción a fórmulas
  • Funciones básicas: SUMA, PROMEDIO, MIN, MAX
  • Funciones lógicas: SI, Y, O
  • Ejercicio práctico: Aplicar funciones básicas para cálculos comunes

📐 Sesión 4: Tablas y Tablas Dinámicas

  • Crear y gestionar tablas
  • Introducción a las tablas dinámicas
  • Análisis básico de datos con tablas dinámicas
  • Ejercicio práctico: Creación de tablas dinámicas para análisis de ventas

📈 Sesión 5: Visualización de Datos con Gráficos

  • Introducción a gráficos
  • Tipos de gráficos: barra, pastel, línea, etc.
  • Personalización de gráficos
  • Formato condicional para resaltar datos
  • Ejercicio práctico: Creación de un reporte gráfico interactivo

🤖 Sesión 6: Automatización Básica y Casos Prácticos

  • Uso de referencias cruzadas y BUSCARV
  • Introducción a macros básicas
  • Casos prácticos:
    • Gestión de inventarios
    • Análisis de presupuesto personal
    • Creación de un reporte de ventas automatizado
